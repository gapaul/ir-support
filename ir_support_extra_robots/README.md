# IR Support Extra Robots

Optional robot model pack for `ir-support`.

This package contains visually useful robot models that are not required for
the core 41013 lab exercises, quizzes, or assignments. Some models were ported
from the old UTS MATLAB toolbox, and some were derived from UTS student
Assignment 2 submissions. Treat student-derived models as useful teaching
assets rather than manufacturer-verified reference models.
